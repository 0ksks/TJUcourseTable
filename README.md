# TJUcourseTable
function:make TJU coursetable

可观看instructions.mp4

steps:

下载makeCourseTable.py文件

用文本编辑打开

将‘’‘中的内容复制到终端并按下回车

登陆1系统

 进入"我的选课-个人课表"页面

在终端中扣1并回车

一份名为courseTable.ics的文件将出现在桌面

打开此文件,课程表将导入日历
